# This is a repository of my config files for Neovim.

**For Linux**:
* Install [ripgrep](https://github.com/BurntSushi/ripgrep)

**For Windows**:
* Install [ripgrep](https://github.com/BurntSushi/ripgrep)
* Install [llvm-mingw](https://github.com/mstorsjo/llvm-mingw)
and add the bin folder to path.
